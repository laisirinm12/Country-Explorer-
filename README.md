# Country Explorer

This is a simple **Country Explorer** web application built using **HTML, CSS and JavaScript**.  
It allows users to search and explore information about countries around the world using public APIs.

The application displays important country details such as the flag, description, capital, population, languages, currency, neighboring countries and a map showing the country's location.

## Features

- Search for any country
- Random country generator
- Country flag display
- Short country description
- Capital, population, area, and region details
- Languages spoken in the country
- Currency information
- Neighboring countries with clickable cards
- Country location displayed using Google Maps
- Autocomplete suggestions while typing country names

## Technologies Used

- HTML5
- CSS3
- JavaScript
- REST Countries API
- Wikipedia API
- Google Maps Embed

## Purpose

This project was created as a front-end exercise to practice:

- Working with external APIs
- Fetching and displaying dynamic data
- DOM manipulation using JavaScript
- Structuring and styling web layouts
- Building interactive UI components
